## How to Run

Linux:

```sh
$ git clone https://github.com/andripwn/instake.git
$ cd instake
$ bash index.bash
```

## Configuration

You can configure the output directory in `index.bash` :

```sh
##### CONFIG #####
useragent=[USER AGENT STRING]
output_dir=[OUTPUT DIRECTORY] # default "." (current directory)
##################
```
